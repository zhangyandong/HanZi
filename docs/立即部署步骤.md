## 🔄 替代方案：GitHub + Vercel

如果命令行部署有问题，可以使用网页部署：

### 方法：通过 GitHub

**1. 推送到 GitHub**
```bash
cd /Users/ts/Desktop/HanZi

# 初始化 Git（如果还没有）
git init
git add .
git commit -m "feat: 汉字学习助手 PWA 版本"

# 创建 GitHub 仓库后
git remote add origin https://github.com/[用户名]/hanzi-learning.git
git push -u origin main
```

**2. 在 Vercel 导入**
1. 访问 https://vercel.com/new
2. 用 GitHub 账号登录
3. 点击"Import Git Repository"
4. 选择你的仓库
5. 点击"Deploy"
6. ✅ 自动部署！

**优势**：
- 以后每次 git push 自动部署
- 网页界面更直观
- 无需命令行

## 🆘 常见问题

### Q: 登录时没收到邮件？
**A**: 
- 检查垃圾邮件文件夹
- 等待1-2分钟
- 或者访问 https://vercel.com 网页登录

### Q: 部署失败？
**A**:
- 确认 `dist/` 目录存在
- 重新构建：`yarn build`
- 再次部署

### Q: 网址无法访问？
**A**:
- 等待几分钟（DNS 传播）
- 检查 Vercel 控制台的部署状态
- 确认部署成功（绿色勾）
