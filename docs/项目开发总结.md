## 🛠️ 技术细节

### 汉字处理
```typescript
// 过滤汉字
filterChinese(text: string): string[]

// 获取拼音（带声调）
getPinyin(char: string): string  // "xué"

// 获取部首
getRadical(char: string): string  // "子"

// 限制字数
limitCharacters(text: string, max: number): string
```

### 笔顺动画配置
```typescript
HanziWriter.create(container, char, {
  width: 500,
  height: 500,
  padding: 20,
  showOutline: true,
  strokeColor: '#FF6B6B',
  strokeAnimationSpeed: 1.5,
  delayBetweenStrokes: 400,
  delayBetweenLoops: 2000,
})
```

### 状态管理
```typescript
interface CharacterContextType {
  inputText: string
  characters: string[]
  currentChar: string | null
}
```
